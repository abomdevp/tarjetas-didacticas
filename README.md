# Tarjetas Didácticas

Esta aplicación se basa en una base de conocimientos con preguntas y respuestas sobre un tema específico y muestra aleatoriamente una tarjeta con la pregunta y varias respuestas. 

## Historias de usuario

Utilice las siguientes historias de usuario para desarrollar esta aplicación

- El usuario puede `ver` una sola carta, elegida al azar.
- El usuario puede `ver` una pregunta en la tarjeta y una lista de cuatro posibles respuestas, cada una identificada por una letra.
- El usuario puede `seleccionar` una respuesta haciendo clic en ella.
- El usuario puede `ver` un mensaje de error cuando se selecciona la respuesta incorrecta.
- El usuario podrá `ver` un mensaje de felicitación cuando seleccione la respuesta correcta.
- El usuario puede hacer clic en el botón `Siguiente pregunta` para ver la siguiente tarjeta didáctica.

## Información adicional
- Para esta aplicación, la base de conocimientos de preguntas y respuestas debe estar codificada en un objeto JavaScript.

## Caracteristicas adicionales:
- El usuario puede hacer clic en un botón de `Resultados` para ver el recuento de respuestas correctas e incorrectas.
- El usuario puede hacer clic en un botón `Reiniciar` para restablecer el recuento de respuestas correctas e incorrectas.
- El usuario puede hacer clic en un botón `Barajar` para volver a generar aleatoriamente la "baraja".
- El usuario puede hacer clic en el botón `Más información` para voltear la tarjeta y ver información adicional. Por ejemplo, información detallada sobre el tema de la pregunta que aparece en el anverso de la tarjeta.
